notd

intentional clipboard capture

Python 3.10+ PowerShell 5.1+ Windows MIT License Zero Dependencies

Copy something. Run notd. It is filed and timestamped. That is it.

click to copy git clone https://github.com/Deze-Tingz/notd.git

Auto Classification

url
Links starting with https:// or www.
command
Terminal commands: git, docker, npm, python...
code
Functions, classes, imports, code blocks
error
Exceptions, tracebacks, stack traces
text
Everything else — notes, snippets, prose

Commands

notd capture Grab clipboard, classify, and save
notd capture --silent Capture without sound feedback
notd status Show config path, data dir, hotkey info
notd open Open captures folder in Explorer
notd config Launch the settings UI
notd hotkey Start global hotkey listener (Ctrl+Alt+N)
